# Waybackrobots
Scrapes URLs from robots.txt of a given url from wayback machine.

# Usage

`python3 waybackrobots.py example.com -o output.txt -t 5`

Flags :

	-o : Output File (Default : robo_domain.txt)
	-t  : Threads ( Default : 10)
